CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan halaman HTML. Contohnya:

```html
<style>
  body {
    background-color: #f0f0f0;
    font-family: sans-serif;
  }

  h1 {
    color: navy;
    text-align: center;
  }
</style>
```
